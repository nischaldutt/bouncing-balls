# Bouncing Balls


Simple game that has a evil circle that eats all the balls.
You have to consume all balls in less time.
HTML, CSS and javascript were used.

## How to Play

- Use 'W' to move forward
- 'A' to move left
- 'D' to move right
- 'S' to move down
